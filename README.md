# نظام إدارة المواعيد الذكي  
Appointment Smart Scheduling System

---

## مقدمة عن المشروع
نظام إدارة المواعيد الذكي هو تطبيق Backend مبني باستخدام Spring Boot، يهدف إلى تنظيم عملية
حجز وإدارة المواعيد للعيادات والمراكز الخدمية والمكاتب الاستشارية.
يساعد النظام على منع التعارض بين المواعيد، والالتزام بأوقات الدوام المحددة،
مع توفير آلية واضحة لإدارة المواعيد من قبل الإدارة.

---

## فكرة المشروع
يسمح النظام للمستخدمين بحجز مواعيد لخدمات محددة ضمن أوقات عمل معينة،
مع دعم عدة أدوار مثل المدير والموظف والعميل.
يمنع النظام حجز المواعيد خارج أوقات الدوام أو تداخلها مع مواعيد أخرى.

---

## أهداف المشروع
- بناء واجهات REST باستخدام Spring Boot  
- تطبيق منطق أعمال واقعي لإدارة المواعيد  
- التعامل الصحيح مع التاريخ والوقت  
- فصل طبقات النظام بطريقة واضحة  
- تأمين النظام باستخدام Spring Security و JWT  
- تصميم قاعدة بيانات قابلة للتوسع  

---

## المستخدمون
- Admin: إدارة النظام والخدمات وأوقات العمل والموافقة على المواعيد  
- Staff: الاطلاع على المواعيد المخصصة له  
- Customer: حجز وتعديل وإلغاء المواعيد  

---

## التقنيات المستخدمة
- Java 17  
- Spring Boot  
- Spring Web  
- Spring Data JPA  
- Spring Security (JWT)  
- H2 Database  
- Maven  

---

## هيكلية النظام
يعتمد النظام على بنية طبقات واضحة:
- Controller Layer  
- Service Layer  
- Repository Layer  
- Entity Layer  

---

## قاعدة البيانات (بشكل عام)
- User: معلومات المستخدم والدور  
- Service: تفاصيل الخدمة  
- WorkingSchedule: أوقات الدوام وأيام العطل  
- Appointment: بيانات الموعد وحالته
